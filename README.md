## 📥 วิธีติดตั้ง

1. ดาวน์โหลดไฟล์ **`Donate-Alert-Setup-1.3.1.exe`** ด้านล่าง
2. ติดตั้งตามขั้นตอน
3. เปิดโปรแกรม → ใส่ **License Key** ที่ได้รับ
4. ตั้งค่าตามวิดีโอสอนด้านบน
